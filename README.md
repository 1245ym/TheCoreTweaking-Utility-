## 🧠 How It Works
The utility dynamically builds a categorized GUI using Windows Forms.  
Each tweak includes:
- A checkbox  
- A description  
- A hover‑based benefit tooltip  
- A safe registry, service, or network command  

Only selected tweaks are applied when you click **Apply Selected Tweaks**.
